Welcome to Slay OS

I like building stuff and breaking things along the way.

What is this?

This is Slay OS — my take on a web-based desktop for the Hack Club Stardance challenge. I built the whole thing from scratch with plain HTML, CSS, and JS. No frameworks, no shortcuts.

🛠️ Built With

HTML + CSS + vanilla JS. That's it. Kept it simple on purpose.

⚡ Cool Stuff

Working terminal with commands, auto-saving notepad, draggable windows, and a login screen.
